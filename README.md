# Nuevas-Tendencias
Desarrollo Front-End
1. Desarrollo Front-End
Instrucción: "Crea una página web simple con HTML y CSS que muestre un mensaje de bienvenida y un botón que cambie el color de fondo al hacer clic."

2. Propósito del Ejercicio
Aplicar prompts efectivos para generar código front-end básico y entender cómo la IA estructura componentes visuales interactivos.

3. Genera el Código Base
Solicita a la IA que cree la estructura HTML, estilos CSS y funcionalidad JavaScript básica para cambiar colores.

4. Analiza el Resultado
Examina cómo la IA organizó el código: estructura HTML semántica, estilos CSS aplicados y event listeners en JavaScript.

5. Modifica Manualmente
Practica editando el código: cambia colores, ajusta tamaños de fuente, agrega animaciones CSS para transiciones suaves.

6. Expande Funcionalidad
Solicita mejoras: "Agrega tres botones que cambien a colores específicos" o "Incluye un selector de paleta de colores".
